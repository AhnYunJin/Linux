
	  
	호스트 운영체제 (Host OS)
		- 물리 PC에 설치된 운영체제
			ex) Windows 7
			
	게스트 운영체제 (Guest OS)
		- 가상 머신(VM)에 설치된 운영체제
			ex) Linux 

	물리 PC 하드 디스크 용량 확인
	내컴픁터
	C:\ (120GB SSD)	
	D:\ (1TB   HDD) 
	여유 공간이 많이 남은 공간에 개인 폴더 생성
	
	C|D:\주말-리눅스-박동삼\VM\Server-A
	
						   
	가상머신 <---> 물리PC
	키보드/마우스 제어 변경 
		Ctrl + Alt

	
	리눅스 관리자 계정
	ID : root 
	PW : password1!
	
	VMware 기능
	
	서버 
	 - 서비스를 제공하는 컴퓨터(프로그램)
	 - 클라이언트에게 서비스를 제공해주는 프로그램
	 - 서버 프로그램이 동작하는 컴퓨터	

	클라이언트 
	 - 서비스를 요청/사용하는 컴퓨터(프로그램)
		
		
			서버 (Server)	<--->	클라이언트 (Client) 
			    관리자					사용자
동작환경		CLI환경에서 동작		GUI환경에서 동작
운영체제		Linux, Unix				Windows, MacOS
				IDC						가정집
				
	처음 리눅스(CentOS)를 설치하면 자동으로 SSH서버용 프로그램(sshd)이
	설치가 되어있으며, 부팅 시 자동으로 활성화되어있다.
		
	SSH클라이언트 프로그램(Xshell)을 이용하여 접속이 가능한 상태이다.
	
	
	리눅스 관련 인물
	- 데니스 리치
		C언어와 Unix를 개발
	- 리누즈 토발즈
		운영체제 핵심인 커널(Kernel)을 개발
		리눅스 == 커널  (시스템 소프트웨어)
	- 리처드 스톨만
		FSF, GNU프로젝트 창단
		GPL라이센스가 적용된 프로그램을 개발(응용 소프트웨어)
		
	GNU + LINUX = GNU/LINUX 운영체제
